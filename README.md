# MADI
Стек MERN
------------------
![MERN](https://miro.medium.com/max/2800/0*QrGJeWvArCjF2BuV.jpg)

### Предворительно должно быть установленно на ПК:
- [Node.js](https://nodejs.org/en/)

### Чтобы подтянут все звисимости надо:
```
npm i
npm run client:install
```

### Для разработки надо:

- Запустить проект 
```
npm run dev
```
